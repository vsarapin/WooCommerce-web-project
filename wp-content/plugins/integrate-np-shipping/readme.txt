== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/integrate-np-shipping` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Proceed to WooCommerce->Settings->Shipping->Novaposhta Shipping screen to configure the plugin
